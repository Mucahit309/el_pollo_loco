# El Pollo Loco 🐔🏜️ (Work in progress)

A fun, object-oriented JavaScript jump-and-run game based on HTML5 Canvas. Help Pepe fight his way through the hot desert, defeat the crazy chickens, collect salsa bottles, and face the ultimate Endboss! 

## 🎮 Play the Game (Work in progress)


## 🚀 Features
* **Object-Oriented Programming (OOP):** Built entirely with JavaScript classes (Character, World, Enemies, Background Objects, etc.).
* **HTML5 Canvas:** Smooth rendering of game graphics and animations.
* **Collision Detection:** Accurate hitboxes for enemies, collectible coins, and throwable salsa bottles.
* **Dynamic Status Bars:** Real-time updates for Pepe's health, collected coins, and salsa bottle inventory.
* **Boss Fight:** A challenging final Endboss that reacts to the player's presence, changes animations upon getting hurt, and takes multiple hits to defeat.
* **Interactive UI:** Custom Start, Game Over, and Win screens integrated directly into the game canvas without overlapping the container.

## ⌨️ Controls
| Key | Action |
| :--- | :--- |
| **◀ / ▶** (Left/Right Arrows) | Move Left / Right |
| **SPACE** | Jump |
| **D** | Throw Salsa Bottle |

*Tip: You can jump on top of the regular chickens to defeat them, but you'll need your salsa bottles for the Endboss!*

## 💻 Local Installation
If you want to run the game locally on your machine or explore the code, follow these steps:

1. **Clone the repository:**
   `git clone https://github.com/YourUsername/el-pollo-loco.git`

2. **Navigate to the project folder:**
   `cd el-pollo-loco`

3. **Open the game:**
   Simply open the `index.html` file in your preferred modern web browser (e.g., Chrome, Firefox, Safari). No build tools or local servers are strictly required!

## 📁 Project Structure
* `/js` - Contains the main initialization scripts (`game.js`, `level1.js`).
* `/models` - Contains all JavaScript classes (e.g., `world.class.js`, `character.class.js`, `endboss.class.js`, `chicken.class.js`).
* `/css` - Styling for the game container and UI overlays.
* `/img` - All image assets (backgrounds, character sprites, UI elements).
